sla-monitor

GitHub: https://github.com/osidenate/sla-monitor

  • Overview
  • Screenshots

This project started as a way for me to measure the quality of my home internet connection. I wanted to monitor the latency between my home server and other computers in real-time.

This part of the system, sla-monitor, is a .NET Console application. It runs on my home server and pings other computers across the internet. It then records the latency to Firebase.

There are three components to this system:

  • sla-monitor: A .NET console application that measures & records latencies. This runs on my home server.
  • sla-web: An AngularJS application that can display the latencies recorded by sla-monitor in real-time.
  • firebase: Stores the configuration for sla-monitor and records the latest latencies.
